# Django Fileserver

Simple fileserver to upload and download files using REST

  - PUT - upload a file
  - GET - to retrieve a file

### Version
- Python: 2.7
- Django: 1.9.4

### Installation

The following modules need to be installed before running the app:
- Django
- djangorestframework

```sh
$ sudo pip install -r requirements.txt
```


### To Run App
```sh
$ ./manage.py runserver
```
